# Mon portfolio

Bienvenue sur mon portfolio !

Ici, vous découvrirez une sélection de projets réalisés dans le cadre de ma formation en développement web ainsi que des expériences professionnelles. Ce portfolio est construit avec React, React Router, et il utilise des animations subtiles grâce à AOS pour rendre la navigation plus dynamique.

J'ai conçu ce site pour mettre en valeur mes compétences en utilisant des technologies modernes comme HTML, CSS, JavaScript, et React. Chaque projet est une démonstration de ma capacité à créer des applications web dynamiques et réactives. Grâce à React Router, la navigation entre les différentes sections de mon portfolio se fait de manière fluide, tandis que les animations AOS ajoutent une touche interactive à l'expérience utilisateur.

Explorez mes réalisations et découvrez mon parcours en développement web, où j'aspire à créer des applications à la fois fonctionnelles, esthétiques et performantes.
